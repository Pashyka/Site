# Site

Статический сайт для публикации через GitHub Pages.

## Структура

- `index.html` — главная страница
- `pages/catalog.html` — страница каталога
- `assets/images/` — изображения
- `docs/` — документы проекта
- `backups/` — резервные копии рабочих файлов

## Запуск локально

Открой `index.html` в браузере или используй любой простой static server.

## Публикация

Проект готов для GitHub Pages (ветка `main`, корень репозитория).
